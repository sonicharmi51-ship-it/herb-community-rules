# Herb • Community Rules

> A public library of `.cursorrules` and AI coding prompts — curated by the [Herb •](https://getherb.in) community.

Herb is a Chrome extension that saves your rules, sessions, and context for AI coding tools like Cursor, Claude, ChatGPT, and Windsurf.

---

## How to use

### Option 1 — Import directly into Herb (one click)
1. Install [Herb •](https://getherb.in) from the Chrome Web Store
2. Open Herb → Community tab → paste any raw file URL from this repo → hit Import

### Option 2 — Copy manually
Click any rule below → copy the raw content → paste into your `.cursorrules` file or AI tool

## How to contribute

1. Fork this repo
2. Add your rule as a `.md` file inside the `rules/` folder
3. Follow the format below
4. Submit a pull request

### Rule format

What this rule does and when to use it.
